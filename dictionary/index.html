HARDWARE_DICTIONARY_v2

BIOSBasic Input/Output System. Firmware that starts the PC.
CMOSBattery that keeps the PC clock and BIOS settings alive.
DIGITIZERThe glass layer of a screen that detects touch.
ESD_STRAPWrist strap used to prevent static damage to parts.
FPC_CONNFlexible Printed Circuit connector (ribbon cable port).
GPUGraphics Processing Unit. Dedicated video processor.
HEATSINKMetal block used to pull heat away from a CPU.
ISOPROPYL90%+ Alcohol used for cleaning thermal paste/motherboards.
JUMP_WIRESmall copper wire used to repair broken board traces.
KICKSTANDThe mechanical support on tablets/consoles.
LI-IONLithium-Ion. The chemistry used in modern phone batteries.
MOTHERBOARDThe main circuit board connecting all components.
NVMENon-Volatile Memory Express. High-speed SSD standard.
OLEDOrganic Light Emitting Diode. High-contrast screen tech.
PENTALOBE5-point star screw used in iPhones and MacBooks.
QUARTZThe crystal used to keep timing on a motherboard.
RAMRandom Access Memory. Temporary storage for active apps.
SPUDGERPlastic prying tool used to avoid short circuits.
THERMAL_PASTECompound that fills gaps between CPU and heatsink.
USB-CUniversal serial bus standard for charging and data.
VRAMDedicated memory for graphics cards.
WATTAGEMeasurement of power consumption for PSUs.
